# Sound Blaster AWE64 Notes

PCem now exposes an AWE64 option in the sound card list. Internally it uses the same emulation core as the AWE32, so functionality and limitations are identical. You still need the `awe32.raw` ROM image for the EMU8000 synth.

The x86Box emulator offers a more complete AWE64 implementation which includes its additional mixer features. If you require accurate AWE64 behaviour, consider comparing against that project.
